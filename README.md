# OceanPulse-Smath-Hacks-26
Monitoring ocean quality is essential for protecting marine ecosystems as well as our own freshwater supply, but traditional monitoring equipment is expensive and not widely accessible. We aim to build a low-cost system that uses multiple sensors to measure water conditions and detect potential environmental changes.
